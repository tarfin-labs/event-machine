### 3. `localqa.md` — New

User-facing guide for when/why to test with real infrastructure. NOT a dev setup script.

```markdown
# Real Infrastructure Testing

## When Unit Tests Aren't Enough

Unit tests with `Machine::test()` cover behavior logic. But some scenarios
require real MySQL, Redis, and Horizon:

- Async child machine completion via queue workers
- Parallel region dispatch via ParallelRegionJob
- Timer sweep via artisan command
- Lock contention under concurrent access
- Forward endpoint HTTP routing

## Prerequisites

MySQL, Redis, and Laravel Horizon running locally.

## Test Structure

Extend `LocalQATestCase`:
- `cleanTables()` — truncates machine tables + drains Redis queues
- `waitFor(callback, timeout)` — polls DB until condition met

## Example: Async Child Delegation

[OrderMachine dispatches PaymentMachine via queue → Horizon processes →
 ChildMachineCompletionJob routes @done → parent reaches completed]

## Example: Parallel Regions

[ShippingMachine has two parallel regions → both complete via Horizon →
 @done fires → parent transitions]

## Gotchas

- Never use Queue::fake() or Bus::fake() in LocalQA tests
- Always cleanTables() in beforeEach
- Use generous timeouts (30-45s) — Horizon startup time
- redis-cli FLUSHALL before test session, NOT between tests
```

### 4. `troubleshooting.md` — New

```markdown
# Testing Troubleshooting

## My guard always fails when using fakingAllGuards()
`fakingAllGuards()` creates Mockery spies. Spies return `null` by default.
In guard evaluation, `null` is falsy → guard fails.
**Fix:** Use `guards:` parameter to set explicit return values.

## Action runs during @always before fakingAllActions() is called
`Machine::test()` runs `getInitialState()` which fires @always transitions.
`fakingAllActions()` in the fluent chain runs AFTER init — too late.
**Fix:** Use `faking:` parameter: `Machine::test(faking: [MyAction::class])`

## withRunningChild() does nothing
It requires database persistence. Doesn't work with `withoutPersistence()`
or `TestMachine::define()`.
**Fix:** Use `Machine::test()` without `withoutPersistence()`.

## simulateChildDone() throws "does not have a child delegation"
You're either: (a) not at the delegating state, or (b) passing the wrong class.
For job actors, pass the job class: `simulateChildDone(MyJob::class, ...)`.

## Context not available to entry actions
`Machine::test(context: [...])` merges context before initialization.
Entry actions see the injected values.

## Timer doesn't fire after startingAt()
`startingAt()` initializes timer tracking. Use `advanceTimers()` as normal.

## Test passes locally but fails in CI
Parallel test execution can cause fake state bleeding between tests.
Add `afterEach` cleanup: `InvokableBehavior::resetAllFakes()`.
```
