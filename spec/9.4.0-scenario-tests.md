## 3. Test Stub Design

### ScenarioTestMachine

Minimal machine covering all 5 state classifications + child machine delegation:

```
idle (TRANSIENT) → @always → routing (TRANSIENT, guarded @always)
  → [IsEligibleGuard=true] → processing (DELEGATION, job: ProcessJob)
     → @done → reviewing (INTERACTIVE)
        → APPROVE → approved (FINAL)
        → REJECT → rejected (FINAL)
        → START_PARALLEL → parallel_check
        → DELEGATE → delegating
     → @fail → failed (FINAL)
     → @timeout → timed_out (FINAL)
  → [IsEligibleGuard=false] → blocked (FINAL)

delegating (DELEGATION, machine: ScenarioTestChildMachine)
  → @done → delegation_complete (FINAL)
  → @done.error → delegation_error (FINAL)
  → @fail → delegation_failed (FINAL)
```

Parallel section (reachable via `START_PARALLEL` from `reviewing`):

```
parallel_check (PARALLEL)
  → region_a:
      initial: checking_a
      checking_a → @done → a_done (FINAL)
  → region_b:
      initial: checking_b
      checking_b → @done → b_done (FINAL)
  → @done → [IsValidGuard] → all_checked (INTERACTIVE)
  → @fail → check_failed (FINAL)
  → SKIP_CHECK → skipped (on-transition on parallel state)
```

### ScenarioTestChildMachine

Child machine with transient initial state (for @start testing):

```
idle (TRANSIENT) → @always → verifying (DELEGATION, job: ProcessJob)
  → @done → [IsValidGuard=true] → verified (FINAL)
  → @done → [IsValidGuard=false] → unverified (FINAL)
  → @fail → child_failed (FINAL)
```

### Guards

```php
class IsEligibleGuard extends GuardBehavior
{
    public static array $requiredContext = ['userId' => 'int'];

    public function __invoke(ContextManager $ctx): bool
    {
        return $ctx->get('eligible', true);
    }
}
```

### Events

```php
class SubmitEvent extends EventBehavior
{
    public static function rules(): array
    {
        return [
            'payload.amount' => ['required', 'integer', 'min:1'],
            'payload.note'   => ['nullable', 'string'],
        ];
    }
}
```
